Getting started

You can now use the learning records service (LRS) to submit your qualification data.

This replaces the RM Results system you may have used in the past.

Getting your file ready

Before you start, you’ll need to prepare your data file using the data specifications.

There are different specifications for general qualification (GQ) data and for vocational and technical qualification (VTQ) data.

The file should be no larger than 100MB and should be in one of the following formats:

  • .csv
  • .xml
  • .json

Testing your template (optional)

You may want to check your template file for any errors before you submit your data, for example if you’ve switched to a learner management system.

This is an optional step and you do not need to do this before making each submission.

Submitting your data

Once you have your data file ready, you can select ‘Start submission’ to upload it for processing. When the processing is complete you’ll be able to check whether your submission has been successful.

Viewing your submission status

Once you’ve uploaded your data file, you’ll be able to view the status of your submission.

If there are no errors in your data, you’ll see a ‘processed successfully’ status so you know that your submission is complete.

If there are errors with your data, you’ll be able to view details of these so you can correct them. You should then upload the corrected file as a new submission.
