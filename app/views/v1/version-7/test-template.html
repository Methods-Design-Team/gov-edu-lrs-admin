Test your template

You can choose to test a template file to make sure you’re ready to submit data, for example, if there’s been a change to your learner management system.

When you upload your template file, we’ll check it for any errors using the same validation as a live file submission. We'll show you the details of any errors in your data, if applicable, but no data will be submitted.

Testing a file is not the same as submitting data. To make a live data submission you should select ‘Make a new submission’ instead.

This is an optional step. You do not need to validate each file that you upload for submission.
